# UvCorny 🌽
A UV Project Management Template for Python.

## Quickstart

Clone this template and work with UV out of the box.

### Default Build Arguments

In .devcontainer/Dockerfile -

- LICENSE : Either MIT or Apache-2.0, default MIT
- LINK_MODE : symlink
- PROJECT_TYPE : --package
- PYTHON_VERSION : 3.12

### ENVIROMENTAL VARIABLES

The Alpine UV image does not have git configured by default.

Set your 

- DEVELOPER : Your GitHub username.
- DEVELOPER_EMAIL : Your Github email.

in your Codespace secret store.
