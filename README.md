# UvCorny
A UV Project Management Template for Python

## Quickstart

If you are using the base Codespace image, it's probably overkill (and slow) for Python package development.

### Build Arguments

In .devcontainer/Dockerfile -

- PYTHON_VERSION : 3.12
- LINK_MODE : symlink

### ENVIROMENTAL VARIABLES

Set in your Codespace Secret Store

Essentials -

- DEVELOPER : Your GitHub username.
- NO_REPLY_EMAIL : Your masked Github email.
